BOX MODEL

De acuerdo con Hong (2018), todos los elementos HTML pueden ser vistos como cajas, el modelo de cajas permite definir los espacios entre ellas.

A continuación se presentan las propiedades de las cajas:

• Contenido: texto, imágenes, etc.
• Borde interno (relleno): área transparente alrededor del contenido, dentro de la caja.
• Margen: espacio entre las cajas.
• Límite: borde alrededor de contenido y relleno. Separa el borde interno del margen.
